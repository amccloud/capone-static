# capone-cli

Quickest way to deploy your static website.

# Installing

```bash
npm install -g capone-cli
```

# Deploying Site

```bash
cd build
capone deploy --site=example
open http://example.caponeapp.com
```

# Custom Domain

```bash
capone deploy --site=example --domain=www.example.com
open http://www.example.com
```
